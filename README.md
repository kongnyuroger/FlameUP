# FlameUP

A lightweight client-side web app that displays latest movies and TV shows using The Movie Database TMDb API. The project includes static HTML, CSS, and modular JavaScript to fetch and render trending, popular, upcoming, top-rated, and TV show data.

## Live demo

View the live demo at: https://flameup.netlify.app/

## Features

- Fetches trending, popular, upcoming, top-rated movies, and popular TV shows.
- Clickable posters show backdrop and overview.
- Simple, responsive static UI built with vanilla HTML, CSS, and ES modules.
- Modular JS functions for each data category (trending, popular, upcoming, top-rated, tv).

## Files of interest

- index.html — main static page and UI shell.
- js/index.js — core fetch/display functions and backdrop logic.
- js/main.js — initializes app and attaches event listeners.
- js/ui.js — UI helpers (empty/placeholder).
